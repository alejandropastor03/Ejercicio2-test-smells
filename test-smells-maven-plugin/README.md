# Test Smells Maven Plugin
Este proyecto implementa un plugin de Maven que analiza los tests de un proyecto
y genera un listado de test smells conocidos.

# Test smells detectados
- Assertion Roulette  
- Conditional Test Logic  
- Constructor Initialization  
- Default Test  
- Duplicate Assert  
- Eager Test  
- Empty Test  
- Exception Handling  
- General Fixture  
- Ignored Test (uso de `@Disabled`)  
- Lazy Test  
- Magic Number Test  
- Mystery Guest  
- Redundant Print  
- Redundant Assertion  
- Resource Optimism  
- Sensitive Equality  
- Sleepy Test  
- Unknown Test
